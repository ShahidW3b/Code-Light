# Code Light – Web-Based Code Highlighting Platform

## 📸 Preview
![Code Light Preview](./assets/images/preview.png)

Code Light is a lightweight, browser-based application developed using **HTML, CSS, and JavaScript**, designed to provide a clean and efficient environment for **syntax highlighting and code visualization**.

The platform enables users to format and preview code snippets in real time, with a focus on readability, simplicity, and modern design principles.

---

## 💻 Core Functionality

Code Light is built to support developers and learners through:

### 🎨 Code Visualization
- Real-time syntax highlighting  
- Clean and structured code presentation  
- Improved readability for complex code snippets  

### 🌗 Theme Customization
- Light and dark mode support  
- Multiple visual styles for better user experience  
- Focus on accessibility and visual clarity  

### ⚡ Interactive Experience
- Instant preview of code input  
- Smooth and responsive UI behavior  
- No installation required (fully browser-based)  

---

## 🛠️ Technologies Used

This project is intentionally developed using **core web technologies** to ensure performance, simplicity, and full control over functionality.

- **HTML5** – semantic structure  
- **CSS3** – responsive and modern styling  
- **Vanilla JavaScript** – dynamic interactivity  

No external frameworks or heavy dependencies are used.

---

## 📱 Features

- Fully responsive design across devices  
- Minimal and user-friendly interface  
- Fast loading and lightweight performance  
- Real-time code rendering
